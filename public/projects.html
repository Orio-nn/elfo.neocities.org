Prototype 1 is a fun codename for a... frankly, right now terrible game I've been working on. The game I've been going for is a Terraria clone, but the one thing holding me back is map generation. I saw a a neat video of someone doing the same type project as me and they used 1d Simplex Noise, but I can't quite figure out how to implement it.
